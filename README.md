# 📚 Smart-AI-Library-Management-System

A professional AI-powered Library Management System developed using **Python, Flask, MySQL, HTML, CSS, Bootstrap, JavaScript and Machine Learning**.

## Features

- User Authentication
- Admin Dashboard
- Student Management
- Book Management
- Issue / Return Books
- Attendance Management
- Fine Management
- Analytics Dashboard
- Machine Learning Book Recommendation
- Book Demand Prediction
- Search Books
- Image Upload
- Responsive UI
